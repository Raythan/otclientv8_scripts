### Passo 2: Como integrar no seu site

Geralmente, sites de Tibia (Gesior/Modern AAC) usam um sistema de `switch` no arquivo `index.php` para carregar as páginas baseadas no parâmetro `?subtopic=...`.

1.  Abra seu `index.php`.
2.  Procure pela parte onde ele verifica o `$_REQUEST['subtopic']` ou `$_GET['subtopic']`. Geralmente parece com:
    ```php
    switch($subtopic) {
        case "latestnews":
            $topic = "latestnews";
            $subtopic = "latestnews";
            include("latestnews.php");
            break;
        // ... outros casos ...
    }
    ```
3.  Adicione o caso para a biblioteca:
    ```php
    case "library":
        $topic = "library";
        $subtopic = "library";
        include("library.php"); // Certifique-se que o caminho está correto
        break;
    ```

### Passo 3: Adicionar o botão no Menu (Sidebar)

Para que os jogadores acessem a página, você precisa adicionar um link no menu lateral esquerdo (onde estão "Community", "Library", "Shop", etc).

No seu arquivo `index.php` (ou `layout.php` dependendo da estrutura), procure a seção `<div id="Menu">`.

Você pode adicionar um novo bloco ou inserir dentro de "Community" ou criar uma aba "Library". Aqui está o código para criar um botão novo igual aos existentes:
